Redirects all feeds to an URI.LV feed and enables realtime feed updates.

== Description ==

URI.LV is the new viable alternative to FeedBurner.

This WordPress plugin automatically handles feeds redirections to your URI.LV feeds the easiest way.

Additionally, every time you publish a new article, a ping is sent to URI.LV to automatically update the feed in realtime.

Coming feature: Ability to create your feed directly from the plugin.

== Installation ==

1. Copy the urilv-feed folder into wp-content/plugins
2. Activate the plugin through the Plugins menu
3. Configure your feed from the new URI.LV Settings submenu
